# Запуск тестов

Проект использует Playwright для E2E-тестирования.

## Команды

```bash
# Запуск тестов с UI-интерфейсом
npm run test:e2e:ui

# Запуск тестов в режиме отладки
npm run test:e2e:debug

# Последовательный запуск (рекомендуется для тестов с аутентификацией)
npm run test:e2e --workers=1
```

## Предварительные требования

Перед запуском тестов убедитесь, что:

1. База данных заполнена сидами: `npm run db:seed`
2. Сервер разработки запущен в отдельном терминале: `npm run dev`
3. Тестовые пользователи существуют (см. [test-users.md](../database/test-users.md))

## Конфигурация

Playwright конфигурация находится в [`playwright.config.ts`](../playwright.config.ts) в корне проекта.

### Основные настройки

- **Базовый URL**: `http://localhost:3000`
- **Тестовые Workers**: по умолчанию включена параллельность
- **Таймауты**: 30 секунд на каждый тест
- **Отчёты**: HTML отчёты сохраняются в `playwright-report/`

## Постоянный режим (watch mode)

Для разработки тестов можно использовать watch mode:

```bash
npx playwright test --watch
```

Это будет автоматически перезапускать тесты при изменении файлов.
