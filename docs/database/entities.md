# База данных

## Основные сущности и связи

Проект использует Prisma ORM с базой данных SQLite. Ниже описаны основные сущности и их связи.

### Диаграмма связей (ERD)

```
┌─────────────────────┐       ┌─────────────────────┐
│        User         │       │        Post         │
├─────────────────────┤       ├─────────────────────┤
│ id (PK)             │◄──────│ authorId (FK)       │
│ email (unique)      │       │ moderatedBy (FK)    │──┐
│ password            │       │ id (PK)             │  │
│ name                │       │ title               │  │
│ slug (unique)       │       │ slug (unique)       │  │
│ role                │       │ content             │  │
│ status              │       │ category            │  │
│ createdAt           │       │ status              │  │
│ updatedAt           │       │ rejectionReason     │  │
└─────────────────────┘       │ created             │  │
         │                     │ moderatedAt         │  │
         │                     └─────────────────────┘  │
         │                              │               │
         │                              │               │
         │                     ┌─────────────────────┐  │
         │                     │   ModerationLog     │  │
         │                     ├─────────────────────┤  │
         │                     │ id (PK)             │  │
         │                     │ postId (FK)         │◄─┘
         │                     │ moderatorId (FK)    │◄─────┘
         │                     │ oldStatus           │
         │                     │ newStatus           │
         │                     │ reason              │
         │                     │ createdAt           │
         │                     └─────────────────────┘
         │
         │
┌─────────────────────┐       ┌─────────────────────┐
│   Notification      │       │       Event         │
├─────────────────────┤       ├─────────────────────┤
│ id (PK)             │       │ id (PK)             │
│ userId (FK)         │ ◄─────┤ authorId (FK)       │
│ title               │       │ moderatedBy (FK)    │
│ message             │       │ title               │
│ isRead              │       │ description         │
│ type                │       │ format              │
│ createdAt           │       │ type                │

                              │ startDate           │

                              │ startRegDate        |

                                        │               ├──────────────────┤
```

### Сущности

#### User (Пользователь)

| Поле          | Тип           | Описание                                             |
| ------------- | ------------- | ---------------------------------------------------- |
| id            | Int           | Уникальный идентификатор (PK)                        |
| email         | String        | Email (уникальный)                                   |
| password      | String        | Хешированный пароль                                  |
| name          | String        | Отображаемое имя                                     |
| slug          | String        | URL-дружественный идентификатор профиля (уникальный) |
| city          | String?       | Город пользователя                                   |
| phone         | String?       | Телефон                                              |
| showContacts  | Boolean       | Показывать контакты в публичном профиле              |
| avatar        | String?       | URL аватара                                          |
| role          | UserRole      | Роль пользователя                                    |
| status        | AccountStatus | Статус аккаунта (ACTIVE, RESTRICTED, BANNED)         |
| emailVerified | DateTime?     | Дата подтверждения email                             |
| createdAt     | DateTime      | Дата создания                                        |
| updatedAt     | DateTime      | Дата последнего обновления                           |

**Роли пользователей (UserRole):**

| Роль       | Описание                                                           |
| ---------- | ------------------------------------------------------------------ |
| USER       | Обычный пользователь (чтение блога, написание постов с модерацией) |
| AUTHOR     | Автор (может создавать и редактировать посты без модерации)        |
| MODERATOR  | Модератор (проверка и модерация постов)                            |
| ADMIN      | Администратор (управление пользователями)                          |
| SUPERADMIN | Суперадминистратор (полный доступ)                                 |

#### Post (Пост блога)

| Поле            | Тип          | Описание                                     |
| --------------- | ------------ | -------------------------------------------- |
| id              | Int          | Уникальный идентификатор (PK)                |
| title           | String       | Заголовок поста                              |
| slug            | String       | URL-дружественный идентификатор (уникальный) |
| excerpt         | String?      | Краткое описание                             |
| content         | String?      | Содержимое в Markdown                        |
| coverImage      | String?      | Обложка поста                                |
| created         | DateTime     | Дата создания                                |
| status          | PostStatus   | Статус публикации                            |
| category        | PostCategory | Категория поста (NEWS, ARTICLE)              |
| viewsCount      | Int          | Количество просмотров                        |
| rejectionReason | String?      | Причина отклонения                           |
| authorId        | Int          | ID автора (FK → User)                        |
| moderatedBy     | Int?         | ID модератора (FK → User)                    |
| moderatedAt     | DateTime?    | Дата модерации                               |

**Статусы поста (PostStatus):**

| Статус    | Описание                                  |
| --------- | ----------------------------------------- |
| DRAFT     | Черновик (редактируется автором)          |
| PENDING   | На модерации (ожидает проверки)           |
| PUBLISHED | Опубликован (доступен в блоге)            |
| REJECTED  | Отклонён модератором                      |
| ARCHIVED  | Архивирован (скрыт из публичного доступа) |

**Категории постов (PostCategory):**

| Категория | Описание             |
| --------- | -------------------- |
| NEWS      | Новости портала      |
| ARTICLE   | Статьи и руководства |

#### Notification (Уведомление)

| Поле      | Тип      | Описание                      |
| --------- | -------- | ----------------------------- |
| id        | Int      | Уникальный идентификатор (PK) |
| userId    | Int      | ID пользователя (FK → User)   |
| title     | String   | Заголовок уведомления         |
| message   | String   | Текст уведомления             |
| isRead    | Boolean  | Прочитано ли уведомление      |
| type      | String   | Тип уведомления               |
| createdAt | DateTime | Дата создания                 |

#### ModerationLog (Лог модерации)

| Поле        | Тип      | Описание                      |
| ----------- | -------- | ----------------------------- |
| id          | Int      | Уникальный идентификатор (PK) |
| postId      | Int?     | ID поста (FK → Post)          |
| eventId     | Int?     | ID мероприятия (FK → Event)   |
| moderatorId | Int      | ID модератора (FK → User)     |
| oldStatus   | String?  | Предыдущий статус             |
| newStatus   | String?  | Новый статус                  |
| reason      | String?  | Причина изменения статуса     |
| createdAt   | DateTime | Дата записи                   |

#### Event (Мероприятие)

| Поле            | Тип         | Описание                                     |
| --------------- | ----------- | -------------------------------------------- |
| id              | Int         | Уникальный идентификатор (PK)                |
| title           | String      | Заголовок мероприятия                        |
| slug            | String      | URL-дружественный идентификатор (уникальный) |
| description     | String?     | Описание                                     |
| format          | EventFormat | Формат проведения (ONLINE, OFFLINE)          |
| type            | EventType?  | Тип мероприятия                              |
| city            | String?     | Город проведения                             |
| location        | String?     | Место проведения (адрес, площадка)           |
| startDate       | DateTime    | Дата и время начала                          |
| endDate         | DateTime    | Дата и время окончания                       |
| startRegDate    | DateTime    | Дата начала регистрации                      |
| endRegDate      | DateTime    | Дата конца регистрации                       |
| coverImage      | String?     | Обложка мероприятия                          |
| status          | EventStatus | Статус публикации                            |
| viewsCount      | Int         | Количество просмотров                        |
| rejectionReason | String?     | Причина отклонения                           |
| authorId        | Int         | ID автора (FK → User)                        |
| moderatedBy     | Int?        | ID модератора (FK → User)                    |
| moderatedAt     | DateTime?   | Дата модерации                               |
| created         | DateTime    | Дата создания                                |
| updatedAt       | DateTime    | Дата последнего обновления                   |

**Форматы мероприятий (EventFormat):**

| Формат  | Описание                    |
| ------- | --------------------------- |
| ONLINE  | Онлайн-мероприятие          |
| OFFLINE | Офлайн-мероприятие (в зале) |

**Типы мероприятий (EventType):**

| Тип         | Описание      |
| ----------- | ------------- |
| MASTERCLASS | Мастер-класс  |
| SEMINAR     | Семинар       |
| KONKURS     | Конкурс       |
| LEKCIYA     | Лекция        |
| VEBINAR     | Вебинар       |
| AREA        | Выставка/AREA |

**Статусы мероприятия (EventStatus):**

| Статус    | Описание                                  |
| --------- | ----------------------------------------- |
| DRAFT     | Черновик (редактируется автором)          |
| PENDING   | На модерации (ожидает проверки)           |
| PUBLISHED | Опубликован (доступен в списке)           |
| REJECTED  | Отклонён модератором                      |
| ARCHIVED  | Архивирован (скрыт из публичного доступа) |

#### EventRegistration (Регистрация на мероприятие)

| Поле         | Тип      | Описание                      |
| ------------ | -------- | ----------------------------- |
| id           | Int      | Уникальный идентификатор (PK) |
| eventId      | Int      | ID мероприятия (FK → Event)   |
| userId       | Int      | ID пользователя (FK → User)   |
| registeredAt | DateTime | Дата и время регистрации      |

#### Favorite (Избранное)

| Поле      | Тип          | Описание                                     |
| --------- | ------------ | -------------------------------------------- |
| id        | Int          | Уникальный идентификатор (PK)                |
| userId    | Int          | ID пользователя (FK → User)                  |
| type      | FavoriteType | Тип записи (POST, EVENT)                     |
| postId    | Int?         | ID поста (FK → Post, если type=POST)         |
| eventId   | Int?         | ID мероприятия (FK → Event, если type=EVENT) |
| createdAt | DateTime     | Дата добавления в избранное                  |

**Типы избранных записей (FavoriteType):**

| Тип   | Описание    |
| ----- | ----------- |
| POST  | Пост блога  |
| EVENT | Мероприятие |

#### Like (Лайк)

| Поле      | Тип      | Описание                      |
| --------- | -------- | ----------------------------- |
| id        | Int      | Уникальный идентификатор (PK) |
| userId    | Int      | ID пользователя (FK → User)   |
| postId    | Int?     | ID поста (FK → Post)          |
| eventId   | Int?     | ID мероприятия (FK → Event)   |
| createdAt | DateTime | Дата лайка                    |

#### PostView (Просмотр поста)

| Поле      | Тип      | Описание                                      |
| --------- | -------- | --------------------------------------------- |
| id        | Int      | Уникальный идентификатор (PK)                 |
| postId    | Int      | ID поста (FK → Post)                          |
| userId    | Int?     | ID пользователя (FK → User, если авторизован) |
| eventId   | Int?     | ID мероприятия (FK → Event)                   |
| views     | Int      | Количество просмотров (по умолчанию 1)        |
| createdAt | DateTime | Дата просмотра                                |

### Связи между сущностями

| Связь                     | Тип         | Описание                                                         |
| ------------------------- | ----------- | ---------------------------------------------------------------- |
| User → Post (author)      | One-to-Many | Один автор может иметь много постов                              |
| User → Post (moderator)   | One-to-Many | Один модератор может проверять много постов                      |
| User → Event (author)     | One-to-Many | Один автор может иметь много мероприятий                         |
| User → Event (moderator)  | One-to-Many | Один модератор может проверять много мероприятий                 |
| User → Notification       | One-to-Many | Один пользователь имеет много уведомлений                        |
| User → ModerationLog      | One-to-Many | Один модератор имеет много записей в логе                        |
| User → EventRegistration  | One-to-Many | Один пользователь может зарегистрироваться на много мероприятий  |
| User → Favorite           | One-to-Many | Один пользователь имеет много избранных записей                  |
| User → Like               | One-to-Many | Один пользователь может поставить много лайков                   |
| User → PostView           | One-to-Many | Один пользователь может просмотреть много постов                 |
| Post → ModerationLog      | One-to-Many | Один пост имеет много записей модерации                          |
| Post → Favorite           | One-to-Many | Один пост может быть в избранном у многих пользователей          |
| Post → Like               | One-to-Many | Один пост может получить много лайков                            |
| Post → PostView           | One-to-Many | Один пост имеет много просмотров                                 |
| Event → EventRegistration | One-to-Many | На одно мероприятие может зарегистрироваться много пользователей |
| Event → ModerationLog     | One-to-Many | Одно мероприятие имеет много записей модерации                   |
| Event → Favorite          | One-to-Many | Одно мероприятие может быть в избранном у многих пользователей   |
| Event → Like              | One-to-Many | Одно мероприятие может получить много лайков                     |
| Event → PostView          | One-to-Many | Одно мероприятие имеет много просмотров                          |

### Жизненный цикл поста

```
┌─────────┐    Отправить на        ┌─────────┐    Одобрить      ┌───────────┐
│  DRAFT  │ ────────────────────► │ PENDING │ ───────────────► │ PUBLISHED │
└─────────┘    модерацию          └─────────┘                  └───────────┘
      ▲                                  │                            │
      │                                  │ Отклонить                  │
      │                                  ▼                            │
      │                            ┌──────────┐                       │
      │                            │ REJECTED │                       │
      │                            └──────────┘                       │
      │                                  │                            │
      │         Вернуть на               │         Архивировать       │
      └──────────────────────────────────┴──────────────────────────►│
                                                                       ▼
                                                                 ┌──────────┐
                                                                 │ ARCHIVED │
                                                                 └──────────┘
```
