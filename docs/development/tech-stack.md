# Технологический стек

Проект построен на современном стеке технологий для разработки веб-приложений на TypeScript.

## Фреймворк и библиотеки

- **Next.js 16** (App Router) — React-фреймворк с серверными компонентами и Server Actions
- **React 19** — UI библиотека
- **TypeScript 5** — строгая типизация
- **Prisma** — ORM для работы с базой данных (SQLite)
- **NextAuth.js** — аутентификация и управление сессиями

## Стилизация

- **Tailwind CSS 4** — утилитарный CSS-фреймворк
- **Ant Design** — библиотека UI компонентов

## Качество кода

- **ESLint 9** — статический анализ кода
- **Prettier** — автоматическое форматирование
- **Playwright** — E2E тестирование

## Логирование

- **Winston** — логирование на сервере

## Инструменты разработки

- **Husky** — Git hooks для автоматического линтинга
- **lint-staged** — запуск линтера только на staged-файлах
