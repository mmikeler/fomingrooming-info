# Информационный портал от грумеров для грумеров

> Информационный портал для грумеров с функциональностью блога, мероприятий и модерации.

## 📚 Документация

Полная документация проекта расположена в папке [`docs/`](./docs/).

### Основные разделы:

- [База данных](docs/database/entities.md) — сущности, связи, ERD-диаграммы
- [Seed данные](docs/database/seed-data.md) — заполнение базы тестовыми данными
- [Тестовые пользователи](docs/database/test-users.md) — учетные данные для разработки
- [Запуск тестов](docs/testing/running-tests.md) — E2E тестирование с Playwright
- [Структура тестов](docs/testing/structure.md) — организация тестов
- [Требования для тестов](docs/testing/requirements.md) — подготовка окружения
- [Просмотр отчётов](docs/testing/report.md) — HTML отчёты Playwright
- [Технологический стек](docs/development/tech-stack.md) — используемые технологии
- [Команды линтинга](docs/development/lint-commands.md) — ESLint и Prettier
- [TypeScript конфигурация](docs/development/typescript-config.md) — настройки TypeScript
- [Правила ESLint](docs/development/eslint-rules.md) — конфигурация линтера
- [Стиль кодирования](docs/development/coding-style.md) — стандарты написания кода
- [Рекомендации для редактора](docs/development/editor-recommendations.md) — VS Code и другие IDE
- [Структура проекта](docs/development/project-structure.md) — организация кодовой базы
- [Блокировка пользователей](docs/security/user-blocking.md) — алгоритмы BANNED/RESTRICTED
- [Форматирование дат](docs/i18n/date-formatting.md) — правила отображения даты

## Установка и запуск

```bash
npm install -save
npm run dev
```

Установка на сервер описана в [Dockerfile](./Dockerfile).

### **0.25.1** 27.04.26

Модернизация формы рекламного блока

- Добавлены обозначения для обязательных полей;
